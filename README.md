# AMD Financial & Stock SQL Analysis
 Overview
This project analyzes the stock performance of Advanced Micro Devices (AMD) using Python, SQL, and financial data analysis techniques.  
It focuses on price trends, volatility, trading volume, and moving averages.

---

Objectives
- Analyze AMD stock price behavior over time
- Build SQL-based financial queries
- Study volatility and momentum
- Apply window functions for time-series analysis
- Create a structured analytics workflow

---

Dataset
- Source: Yahoo Finance (via yfinance)
- Company: AMD (Advanced Micro Devices)
- Time Period: 2022–2026

---

 Tools Used
- Python
- Pandas
- SQLite
- SQL (Window Functions)
- Matplotlib
- Google Colab


 SQL Analyses
 1. Average Closing Price
Calculates AMD’s average stock price over the dataset.
 2. Moving Average (30-day)
Smooths price data to identify trends.

3. Volume Analysis
Tracks trading activity and liquidity trends.

. Price Change Analysis
Measures daily price movement using LAG function.

insights

- AMD shows clear cyclical volatility driven by tech market trends
- Moving averages reveal long-term upward momentum phases
- Trading volume spikes often align with high volatility periods
- Price changes show clustering during major market events
